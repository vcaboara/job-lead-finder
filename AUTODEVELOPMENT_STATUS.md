### Option A: Human-Supervised Execution
Since we can't directly invoke Gemini from this context, you'll need to:

1. **Start Services:**
   ```powershell
   docker compose up -d
   ```

2. **Take BEFORE Screenshot:**
   ```powershell
   Start-Process "http://localhost:8000"
   ```
   Screenshot the current UI (no logos)

3. **Execute Task with Gemini:**
   - Open `.ai-tasks/track-8-gemini-logo-ui.md`
   - Copy task to Gemini interface (web or API)
   - Have Gemini generate the code

4. **Apply Changes:**
   - Gemini will provide HTML/CSS modifications
   - Apply to `src/app/templates/index.html`

5. **Test & Screenshot:**
   ```powershell
   docker compose up --build -d
   Start-Process "http://localhost:8000"
   ```
   Screenshot the new UI (with logos)

6. **Create Branch & PR:**
   ```bash
   git checkout -b auto/ui-company-logo-placeholders
   git add src/app/templates/index.html CHANGELOG.md
   git commit -m "feat: Add company logo placeholders"
   git push -u origin auto/ui-company-logo-placeholders
   gh pr create --title "feat: Add company logo placeholders" --body "<screenshots>"
   ```
